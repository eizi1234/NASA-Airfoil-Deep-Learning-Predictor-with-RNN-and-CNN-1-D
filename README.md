# NASA Airfoil Self-Noise Deep Learning Predictor & Web App

## Project Overview
This repository contains an end-to-end Machine Learning and Deep Learning project developed as part of my coursework for **TF4063 Engineering Data Science** at **Institut Teknologi Bandung (ITB)**. 
I have migrated this academic project to GitHub and deployed it as an interactive web dashboard to showcase my capabilities in software engineering, physical system modeling, and data science architecture.
The core objective of this project is to model and predict the **Scaled Sound Pressure Level (SSPL)**, this representss aerodynamic noise generated by NACA 0012 airfoil blades inside a wind tunnel-based on five critical physical parameters.

## Implemented Methodologies & Domain Knowledge
This project bridges **Engineering Physics (Aeroacoustics & Fluid Dynamics)** with advanced **Data Science / Deep Learning Frameworks**. The specialized skills and concepts implemented include:

### 1. Physics & Engineering Domain (Aeroacoustics)
* **Aeroacoustic Characterization:** Understanding the physical correlation between fluid stream turbulence over a lifting surface and its acoustic sound pressure radiation.
* **Multivariable Physical Features:** Manipulating experimental configurations consisting of Frequency ($f$), Angle of Attack ($\alpha$), Chord Length, Free-stream Velocity, and Boundary-layer Suction Displacement Thickness ($\delta$).

### 2. Deep Learning Architectures (`TensorFlow` & `Keras`)
* **Instantaneous Non-Linear Regression (1D-CNN):** Built a One-Dimensional Convolutional Neural Network utilizing $1 \times 1$ kernels to automatically extract abstract structural relationships and cross-feature correlations across the 5 input physical parameters.
* **Sequential Feature Forecasting (SimpleRNN & LSTM):** Formulated a temporal look-back strategy using a 10-step sliding window to predict acoustic response trends based on sequential index configurations.

### 3. Data Engineering & Lifecycle Pipeline
* **Robust Preprocessing:** Handled features through systematic outlier checking, structured data normalization using `MinMaxScaler`, and multidimensional matrix reshaping to fit deep neural network input standards.
* **Performance Auditing:** Computed rigorous multi-metric evaluations including Mean Squared Error (MSE), Mean Absolute Error (MAE), and Coefficient of Determination ($R^2$ Score) alongside model training execution speed benchmarks.
* **Production-Grade Serialization:** Implemented object serialization techniques using `.keras` and `.pkl` formats via `joblib` for model packaging and runtime reuse.

### 4. Full-Stack Web App Deployment
* **Streamlit Architecture:** Developed a unified, single-script production interface that manages multi-physical slider controllers, inputs real-time raw data, runs background deep learning model inferences, and automatically renders dynamic data analytics and correlation charts (`Seaborn`/`Matplotlib`).
